# UniPhysics Tutor 演示问题

以下 10 个演示问题涵盖了力学和电磁学的核心概念，适合向老师或评估者展示系统能力。

---

## 场景一：本科概念讲解

### 演示问题 1

**问题：** 为什么高斯定理总是成立，但不一定能直接求出电场？

**演示重点：**
- 区分"定理成立"和"求解方便"
- 强调对称性（球对称/圆柱对称/平面对称）
- 给出积分形式和微分形式
- 指出常见误区：学生往往认为高斯定理只在对称时成立

**预期回答亮点：**
```
高斯定理本身是普遍成立的，是库仑定律和叠加原理的推论。
但用它直接求电场需要能把 E 从积分号中提出来，
这通常只有在电荷分布具有球对称、圆柱对称或平面对称时才能做到。
```

---

### 演示问题 2

**问题：** 保守力为什么可以定义势能？

**演示重点：**
- 环路积分为零 → 做功与路径无关 → 可以定义状态函数
- 势能差 = 保守力做功的负值
- F = -∇U 的关系
- 反例：摩擦力不能定义势能

**预期回答亮点：**
```
因为保守力对任意闭合路径做功为零（∮F·dr = 0），
这意味着保守力做功只与初末位置有关，与路径无关。
因此可以定义一个只依赖于位置的状态函数——势能 U(r)，
使得 ΔU = -W_conservative，且 F = -∇U。
```

---

### 演示问题 3

**问题：** 速度为零时加速度一定为零吗？

**演示重点：**
- 竖直上抛最高点：v=0, a=g
- 速度和加速度是两个独立的概念
- 速度是位移的变化率（位置求导），加速度是速度的变化率（速度求导）
- SHM 端点也是 v=0 但 a 最大的例子

**预期回答亮点：**
```
错误。
速度描述位置变化的快慢，加速度描述速度变化的快慢。
竖直上抛物体在最高点速度为零，但加速度始终为 g。
速度为零只能说明位移在该瞬间不变化，
不能说明速度本身不变化。
```

## 场景二：分步解题

### 演示问题 4

**问题：** 一质量为 m 的物体在倾角为 θ 的光滑斜面上下滑，求加速度。

**演示重点：**
- 建立坐标系（沿斜面和垂直斜面）
- 受力分析图
- 分解重力：mg sinθ 和 mg cosθ
- 牛顿第二定律分量形式
- 结果 a = g sinθ，与质量无关
- 单位检查：m/s²

**预期回答亮点：**
```
1. 选坐标系：x 轴沿斜面向下，y 轴垂直斜面向上
2. 受力：重力 mg(↓)，支持力 N(↗⊥斜面)
3. 分解重力：F_x = mg sinθ, F_y = -mg cosθ
4. y 方向平衡：N = mg cosθ
5. x 方向：mg sinθ = ma → a = g sinθ
6. 注意：结果与质量 m 无关！
```

---

### 演示问题 5

**问题：** 一质量为 m 的小球从高度 h 处无初速度下落，忽略空气阻力，用两种方法求落地速度。

**演示重点：**
- 方法一：运动学公式 v² = 2gh
- 方法二：机械能守恒 mgh = ½mv²
- 两种方法结果一致
- 强调机械能守恒条件（只有保守力做功）
- 指出与质量无关

## 场景三：误区诊断

### 演示问题 6

**学生说法：** "电势为零的位置，电场也一定为零。"

**演示重点：**
- 判断：错误
- 电场是电势的空间梯度 E = -∇V
- V=0 不代表 ∇V=0
- 举例：两点电荷连线中点 V=0 但 E≠0

**预期回答亮点：**
```
这个说法是错误的。
电场是电势的负梯度：E = -∇V = -(∂V/∂x, ∂V/∂y, ∂V/∂z)。
某点电势为零，只说明该点的标量电势值为零，
不能说明其空间变化率（即梯度）为零。
经典反例：等量异号两点电荷连线的中点电势为零，但电场不为零。
```

---

### 演示问题 7

**学生说法：** "任何碰撞过程中机械能都是守恒的。"

**演示重点：**
- 判断：错误
- 只有完全弹性碰撞机械能才守恒
- 非弹性碰撞有动能损耗（转化为内能/形变能）
- 动量守恒在碰撞中几乎总是成立（内力 >> 外力）
- 完全非弹性碰撞：机械能损失最大

## 场景四：引导式教学

### 演示问题 8

**问题：** 一个粒子以速度 v 垂直进入匀强磁场 B，为什么速率不变但方向改变？

**引导流程：**
1. **提示 1：** "你能先判断洛伦兹力方向和速度方向的关系吗？"
2. **提示 2：** "如果力始终垂直于速度，这个力做功吗？"
3. **提示 3：** "不做功意味着动能不变，速率不变。那为什么粒子不是直线运动？"
4. **完整解析：** 洛伦兹力 F = qv×B 始终垂直于 v，P = F·v = 0 → 不做功 → 速率不变。但 F≠0 → 有加速度 → 方向改变，形成匀速圆周运动，半径 r = mv/qB。

---

### 演示问题 9

**问题：** 什么情况下机械能守恒？请判断：一个木箱在粗糙水平面上被推动，机械能守恒吗？

**引导流程：**
1. **提示 1：** "先列出有哪些力在做功。"
2. **提示 2：** "摩擦力是保守力还是非保守力？"
3. **完整解析：** 有摩擦力（非保守力）做功 → 机械能不守恒。部分动能转化为内能。机械能守恒要求系统内只有保守力（重力、弹力、静电力等）做功，非保守力做功必须为零。

---

### 演示问题 10

**问题：** 楞次定律中的"阻碍变化"到底是什么意思？能否举个具体例子？

**演示重点：**
- 阻碍的是"磁通量的变化"，不是"磁通量本身"
- 磁铁靠近线圈：B 增大 → 感应电流产生反向 B（阻碍增大）
- 磁铁远离线圈：B 减小 → 感应电流产生同向 B（阻碍减小）
- 楞次定律的深层原因：能量守恒

## 演示建议

### 给老师展示时的流程

1. **先展示概念讲解（问题 1-3）**：让老师看到系统如何拆解概念、指出误区
2. **再展示解题（问题 4-5）**：展示分步推导、公式渲染、单位检查
3. **展示误区诊断（问题 6-7）**：主动输入学生的错误说法，看系统如何诊断
4. **展示引导模式（问题 8-10）**：切换为引导模式，展示交互式教学

### 强调要点

- 回答不是简单的知识复述，而是有教学逻辑
- 公式通过 KaTeX 渲染，板书区专门展示推导
- 每个回答都会检查适用条件、指出常见误区
- 校验器会在后台自动检查物理严谨性
- 知识库检索（RAG）确保答案有教材依据
