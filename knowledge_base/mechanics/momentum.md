# 动量与角动量

## 核心结论

动量定理将力的时间累积效应（冲量）与动量变化联系起来。动量守恒是自然界最基本的守恒律之一，比牛顿第三定律适用范围更广。

## 数学表达

**动量定义：**
\[
\mathbf{p} = m\mathbf{v}
\]

**冲量-动量定理：**
\[
\mathbf{J} = \int_{t_i}^{t_f} \mathbf{F} dt = \Delta\mathbf{p} = \mathbf{p}_f - \mathbf{p}_i
\]

**动量守恒（系统合外力为零）：**
\[
\sum \mathbf{p}_i = \text{constant}
\]
即
\[
m_1\mathbf{v}_{1i} + m_2\mathbf{v}_{2i} = m_1\mathbf{v}_{1f} + m_2\mathbf{v}_{2f}
\]

**角动量定义：**
\[
\mathbf{L} = \mathbf{r} \times \mathbf{p} = \mathbf{r} \times m\mathbf{v}
\]

**力矩：**
\[
\boldsymbol{\tau} = \mathbf{r} \times \mathbf{F}
\]

**角动量定理：**
\[
\sum \boldsymbol{\tau} = \frac{d\mathbf{L}}{dt}
\]

**角动量守恒：**
当合外力矩为零时，\( \mathbf{L} = \text{constant} \)

**碰撞分类：**
- 完全弹性碰撞：动能守恒 + 动量守恒
- 完全非弹性碰撞：碰撞后粘在一起，动能不守恒，动量守恒
- 非完全弹性碰撞：动量守恒，恢复系数 \( 0 < e < 1 \)

## 适用条件

- **动量守恒**：系统所受合外力为零，或内力远大于外力（如碰撞过程）
- **角动量守恒**：系统所受合外力矩为零
- 动量定理是矢量关系，需注意分量守恒（如某方向合外力为零，则该方向动量分量守恒）
- 动量守恒比牛顿力学适用范围更广，在量子力学和相对论中也成立

## 常见误区

1. **任何碰撞机械能都守恒**：只有完全弹性碰撞机械能才守恒。多数碰撞有能量损耗。
2. **动量守恒需要系统"封闭"**：只需合外力为零，内部可以有相互作用。
3. **忘记动量是矢量**：一维问题中需注意正负号（方向），二维问题中需分解为分量处理。
4. **把动量守恒和动能守恒混淆**：两者是独立的守恒律，条件完全不同。
5. **角动量只对转动有意义**：质点直线运动也有角动量，取决于参考点。
6. **碰撞问题中把速度的正负号搞反**：设定正方向后，反向运动的速度必须带负号。

## 典型例题

**题目：** 质量为 \( m_1 \) 的子弹以速度 \( v \) 射入质量为 \( m_2 \) 的静止木块并留在其中（完全非弹性碰撞），求共同速度。

**解答：**

碰撞过程内力远大于外力，动量守恒：
\[
m_1 v + 0 = (m_1 + m_2) v'
\]
\[
v' = \frac{m_1}{m_1 + m_2} v
\]

动能损失：
\[
\Delta K = \frac{1}{2}m_1 v^2 - \frac{1}{2}(m_1 + m_2)v'^2 = \frac{1}{2}\frac{m_1 m_2}{m_1+m_2}v^2
\]
动能不守恒（转化为内能）。

## 教学提示

- 强调动量守恒条件是"合外力为零"而非"没有力"
- 碰撞问题中动量守恒几乎总是成立，但机械能守恒需要额外判断
- 教会学生"先判断用哪个守恒律，再列方程"
